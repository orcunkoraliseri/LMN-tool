Layer 0: Output Page Documentation

Overview

The Output Page (layer0_output.html) displays the matching neighbourhoods based on selections from Layer 0.

File Path: /Interface/layer0_output.html

Features

  • Dynamic Filtering: Loads sessionStorage filters to display specific matching entries from data.js.
  • Results Table: Displays rows with Concept images, 3D layouts, and neighbourhood metrics.
  • EUI scale: Color-coded energy performance scale (0-100 range).
  • Building Mix: Composition of building types using specialized icons.

Technical Logic

Controlled by js/app.js or js/output.js logic which iterates through NEIGHBOURHOODS data, comparing metadata against selected filters.
